Задание 2.1
Для работы банкомата мы будем использовать паттерн "Состояние". Смотри Exercise_2.1.png.
В качестве самого банкомата мы будем использовать Context.
У банкомата может быть множество состояний которые будут описываться в - ConcreteState'ах:
	карта не вставлена — отклонить попытку ввода PIN-кода
	PIN-код введен — разрешить операции
	карта извлечена — снова перейти в исходное состояние
	введено доступное кол-во денег - снять деньги с банковского счета

Задание 2.2
Для реализации сохранений действий с возможностью отменять/повторять действия графическом редакторе
мы будем использовать паттерн "Хранитель" Смотри Exercise_2.2.png.

Memento это хранитель который хранит в себе действия пользователя.
Originator это пользователь, он же инициатор.
Caretaker это опекун который выполняет функцию хранения объекта Memento.

Задание 2.3
Для инициализации десятков объектов врагов с одинаковыми характеристиками целесообразно использовать
паттерн "Прототип". Смотри Exercise_2.3.png.
Данный паттерн содержит метод Clone() который можно использовать во время "заполнения" локации
одинаковыми видами врагов.
